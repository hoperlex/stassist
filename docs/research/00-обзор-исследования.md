# 00. Обзор исследования

> Пакет исследования портала «Stassist» · дата среза: 12.07.2026

## Цель

Изучить сектор астрологических услуг и знаний, рынок существующих порталов/приложений
(мировых и русскоязычных), востребованные услуги, школы, расчётные алгоритмы, модели
монетизации, юридические ограничения и SEO-ландшафт — как основу для проектирования
коммерческого русскоязычного портала.

## Методика

1. **Скаутинговое исследование** — 8 параллельных агентов-исследователей, по одному на
   направление; каждый выполнил 8–15 веб-поисков (рус./англ.) и чтение первоисточников.
   Итого 223 веб-действия, ~160 источников.
2. **Каталог топ-50** — 50 агентов-профилировщиков (по одному на конкурента, профиль в JSON
   по фиксированной схеме → `_work/top50/`), затем 12 ключевых профилей проверены
   агентами-скептиками (`_work/top50/_verify/`). Итог: [09-топ50-каталог.md](09-топ50-каталог.md)
   и Excel-матрица `_report/top50-матрица.xlsx` (собрана скриптом из JSON — детерминированно).
3. **Адверсариальная верификация** — 8 агентов-скептиков перепроверили по первоисточникам
   критичные для проектных решений факты (лицензии, 152-ФЗ, правила рекламы, рыночные
   метрики, цены, показатели конкурентов, НПД/платформы, SEO-факты) → `_work/verification/`.

Правила достоверности (по стандартам рабочего пространства): каждое количественное
утверждение снабжено источником; оценки трафика/рынка помечены «оценка»; непроверенное —
«требует проверки»; пробелы не заполняются догадками.

## Главные выводы (сводно)

1. **Ниша ИИ-астрологии по подписке в рунете свободна** — доминирующего игрока нет; лидеры
   рунета (horo.mail.ru и клоны) — рекламные медиапорталы без персонализации уровня Co-Star.
2. **Работающая экономика**: бесплатные калькуляторы как лидоген → разовые PDF-отчёты →
   подписка на персональную ленту → (этап 2) маркетплейс консультаций.
3. **Расчёт всегда детерминированный, LLM только интерпретирует** — консенсус индустрии
   против галлюцинаций; собственный корпус интерпретаций — дефицитный актив.
4. **Swiss Ephemeris недоступен для закрытого SaaS без платной лицензии** (AGPL-3.0);
   open-source путь — astronomy-engine (MIT) + собственные системы домов.
5. **Главный канал роста — SEO-программатика и собственные каналы**: платная реклама
   эзотерики жёстко ограничена площадками и находится под регуляторным риском.
6. **152-ФЗ**: данные рождения — персональные данные; отдельное согласие, локализация в РФ
   (конфликт с облачным Supabase — решение в техархитектуре).
