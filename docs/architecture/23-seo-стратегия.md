# 23. SEO-стратегия и каналы продвижения

> Пакет документов портала «Stassist» · 12.07.2026
> Основание: [08-seo-и-коммьюнити.md](../research/08-seo-и-коммьюнити.md). Оценки частотностей
> требуют съёма в Яндекс Wordstat на этапе запуска (доступ — ручной, помечено в плане).

## 1. Почему SEO — главный канал

Платная реклама эзотерики ограничена: в Директе гадания/ворожение допускаются лишь с
ограничениями (запрет гарантий, урезанные форматы), VK допускает только «чистую астрологию»
без обещаний результата; первая версия законопроекта о запрете рекламы эзотерики отклонена
(отзыв правительства, июль 2025), но **13.02.2026 внесена новая редакция** (первое чтение) —
риск актуален, мониторить до запуска (см. поправки в [05](../research/05-рынок-услуги-юридика.md)). Лидеры ниши живут органикой (Rambler ~42% органики;
horo.mail.ru ~16,9 млн кликов/мес из Яндекса — оценки). Вывод: трафик строим на программатике,
контенте и собственных каналах; реклама — вспомогательно.

## 2. Программатическая матрица (генерируется системой)

| Кластер | Шаблон URL | Объём | Обновление |
|---|---|---|---|
| Гороскопы | `/goroskop/{znak}/{period}` (+`/lyubov|finansy|karera|zdorove`) | 12×5×5 = 300 | день/неделя/месяц/год |
| Годовые | `/goroskop/2027/{znak}` (+восточный `/vostochnyj-goroskop/2027/{zhivotnoe}`) | 24/год | сезонно, публикация к октябрю |
| Совместимость | `/sovmestimost/{znak-a}-i-{znak-b}` | 78 (зеркала — 301) | статично + виджет расчёта |
| Лунный календарь | `/lunnyj-kalendar/{yyyy-mm}` + `/lunnyj-den/{n}` + тематические `/strizhka|posadki|manikyur/{yyyy-mm}` | ~50/мес | ежедневный автопересчёт |
| Натальная карта | `/natalnaya-karta` (калькулятор) + `/planety/{planeta}-v-{znak|dom}` | 10×24 = 240 | статично из вики-чанков |
| Матрица судьбы | `/matrica-sudby` (калькулятор) + `/arkan/{1..22}` | 23 | статично |
| Камни | `/kamni/{kamen}` + `/kamni-po-znaku/{znak}` | 60–80 на старте (цель ~150) + 12 | статично |
| Вики | `/wiki/{razdel}/{slug}` | 200+ на старте | редакционно |
| Знаменитости | `/karta/{celebrity}` | 500+ (постепенно) | из базы celebrities |

Правила: каждая страница — уникальный H1/title/description по шаблонам с вариациями;
внутренняя перелинковка как рекомендательная система (гороскоп → совместимость → калькулятор →
вики); страницы с «протухающим» контентом (день/неделя) обновляются worker'ом строго по
расписанию — стухшая программатика деранжируется.

## 3. Техническое SEO

- **SSR** для всего публичного контура (vike поверх Fastify, [21, §2](21-техническая-архитектура.md)).
- JSON-LD: `Article`/`NewsArticle` (гороскопы, статьи), `BreadcrumbList`, `Organization`,
  `WebSite`+`SearchAction`; FAQPage/HowTo не используем (Google их не показывает с 2023).
- Sitemap-индексы по кластерам (гороскопы/календарь/вики/камни), lastmod честный; robots с
  запретом кабинета/API; канонизация (зеркала совместимости, UTM).
- Core Web Vitals: SSR + минимальный JS на программатике (hydration по требованию —
  интерактивные виджеты как острова); бюджет Lighthouse в CI.
- Региональность Яндекса: единый контент без geo-страниц на MVP (услуги онлайн);
  Яндекс.Вебмастер + turbo не используем (устарел), микроразметка контактов.
- OG-карточки: генерация PNG (resvg в worker) для карт, совместимостей, гороскопов дня —
  CTR в соцсетях/мессенджерах.

## 4. Поведенческие факторы (критично для Яндекса)

Интерактив на посадочных: калькулятор прямо на программатической странице (наталка на
`/natalnaya-karta`, виджет пары на страницах совместимости, «какой у меня лунный день» на
календаре); «астропогода сегодня» — виджет на всех страницах гороскопов. Цель: глубина >2,
время >2 мин — уровень, недостижимый для статических «ферм».

## 5. E-E-A-T (для Google) и доверие

Страницы авторов/редакции (редактор-астролог с биографией), методология расчётов
(`/methodology` — версия ядра, точность, источники алгоритмов; уникально для ниши),
дисклеймер и политика ПД в футере, дата обновления на статьях, ссылки на первоисточники в вики.

## 6. Семантика: план съёма (задача этапа запуска)

Ядро собирается вручную в Wordstat (API недоступен без спецдоступа): кластеры «гороскоп+знак+период»,
«натальная карта [онлайн|с расшифровкой|бесплатно]», «совместимость знаков», «лунный
календарь [стрижек|посадок|маникюра] [месяц год]», «матрица судьбы [рассчитать|расшифровка]»,
«камень [знака]», «[планета] в [знаке|доме]», «ретроградный меркурий [даты год]».
Ориентиры из исследования (оценки, требуют съёма): «натальная карта» ~930 тыс. показов/мес;
сезонный пик «гороскоп [год]» — ноябрь–декабрь; «матрица судьбы» — сопоставимо с наталкой.
Инструментальная проверка: Wordstat вручную + Keys.so/Bukvarix для конкурентного среза.

## 7. Контент-план (первые 3 месяца после запуска)

1. Вики-ядро: 24 статьи «планета в знаке» ×10 планет (генерация из чанков + редактура) —
   двойное назначение (SEO + RAG).
2. Сезонное: страницы «гороскоп 2027» (сентябрь–октябрь 2026), «год Огненной Лошади».
3. Инфоповоды: ретро-Меркурий (даты и «что делать»), затмения, полнолуния — шаблонные
   лендинги с датами из astro_calendar.
4. Долгий хвост: разборы карт знаменитостей (UGC + редакция).

## 8. Собственные каналы

- **Telegram**: канал-сетка по знакам (12 мини-каналов + главный) — автопостинг из пайплайна
  гороскопов; бот с персональным прогнозом (вход по ссылке с портала, deep-link в кабинет).
  Экономика набора: подписчик 20–25 ₽, CPM 400–450 ₽ (оценки vc.ru).
- **Дзен**: 3–5 публикаций/нед из редакционного контента (алгоритм не требует базы).
- **Email**: транзакционные + еженедельный дайджест «ваша астронеделя» (персонализация из ядра).
- **Пуш** (web-push + Telegram): персональный прогноз утром, точные транзиты («сегодня ваш
  соляр», «Меркурий выходит из ретро») — тон дружелюбный, без запугивания.

## 9. Метрики SEO

Позиции по 200 контрольным запросам (еженедельно), органические визиты по кластерам,
индексация программатики (покрытие в Вебмастере/GSC), CTR сниппетов, конверсия
посадочная→калькулятор→регистрация. Дашборд — Metabase поверх PostgreSQL + Метрика/GSC.
