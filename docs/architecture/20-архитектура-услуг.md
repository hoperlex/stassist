## M0. Профиль пользователя и карты (фундамент)

- **Данные рождения**: дата, время (с флагом «время неизвестно»), место (геокодинг → lat/lon/TZ,
  исторические часовые пояса через tz-базу). Несколько профилей на аккаунт («я», «партнёр»,
  «ребёнок», «знаменитость» из встроенной базы).
- **Расчётный профиль** (пресет школы): зодиак (тропический/сидерический+айанамша), система домов,
  набор объектов, орбисы. Пресеты: «Современная западная» (по умолчанию), «Классическая»,
  «Джйотиш» (версия 1.x), «Школа Глобы» (версия 2+ — требует фиктивных точек); тонкая настройка — для Pro.
- «Время неизвестно» → карта без домов/Asc, честная деградация функций (без солнечных домов «на глаз»).
- Из профиля деривируются все карты: натал, транзиты, прогрессии, соляр, синастрия.

## M1. Гороскопы (ТЗ-1) — SEO-ядро

- **Программатическая сетка**: 12 знаков × {сегодня, завтра, неделя, месяц, год} ×
  {общий, любовь, финансы, карьера, здоровье} + восточный годовой + лунный день.
- **Производство контента**: ежедневный пайплайн — расчёт реальных транзитов дня (астроядро) →
  структурированные «астрособытия» (Луна в знаке, аспекты дня, void, ретро-статусы) → LLM
  генерирует тексты по шаблонам тона → премодерация (автофильтры + выборочная ручная) →
  публикация по расписанию. Тексты привязаны к реальному небу — отличие от «фермерских» гороскопов.
- **Персональный гороскоп** (залогиненным): транзиты к натальной карте, приоритизация точных
  аспектов; краткая версия бесплатно, полная лента — подписка.
- Виджеты «астропогода сегодня»: Луна в знаке, лунный день, void-периоды, ретроградные планеты.

## M2. Шуточные гороскопы (ТЗ-2) — виральный контур

- Форматы: «антигороскоп» (ироничный), профессиональные («гороскоп разработчика», «гороскоп
  бухгалтера»), мемные подборки; генерация LLM с отдельным тоном, редакционная витрина.
- Шеринг-карточки (OG-изображения с текстом) — каждый выпуск шерабелен; отдельная лента в Telegram.
- Чёткая маркировка «шуточный» — не смешивается с основным контентом (доверие + юридическая чистота).

## M3. Индивидуальные прогнозы под заказ (ТЗ-3)

- **Мастер заказа**: тип (событие / период / вопрос-план) → данные (дата события, интервал,
  формулировка) → выбор глубины (краткий/полный) → оплата → генерация.
- **Расчётная база по типу заказа**: событие → транзиты/лунный контекст на дату + соотнесение
  с наталом; период → транзитная карта периода, ключевые точные аспекты, соляр/лунар если
  покрывается; план/вопрос → элективные окна (благоприятные даты в интервале по заданным
  критериям: аспекты к наталу, фаза Луны, void, ретро).
- **Выдача**: онлайн-просмотр + PDF (брендированный, с картой-SVG, дисклеймером и списком
  расчётных данных «как считали» — прозрачность = доверие).
- **Задел этапа 2**: заказ имеет статусную модель (создан→оплачен→выполнен ИИ→[передан
  астрологу→принят→выполнен]) — маркетплейс встраивается без перестройки.

## M4. ИИ-астропомощник (ТЗ-4)

- **Чат с контекстом карты**: пользователь спрашивает («что у меня с карьерой в августе?»),
  бэкенд сам считает нужные карты и передаёт LLM структурированные данные + RAG-фрагменты.
  LLM никогда не вычисляет позиции (архитектура — [21](21-техническая-архитектура.md), §4).
- **Готовые разборы** (генерируются по кнопке, кэшируются): полный разбор натала (по сферам:
  личность, отношения, карьера, ресурсы), синастрия пары, соляр года, транзитный прогноз на месяц,
  джйотиш-разбор (версия 1.x), разбор Матрицы судьбы.
- **Freemium-границы**: бесплатно — Big Three-разбор, 3 вопроса чату/день, краткие разборы;
  подписка — безлимитный чат (fair use), полные разборы; разовые PDF — без подписки.
- **Гарантии качества**: каждый разбор содержит блок «Расчётные данные» (позиции/аспекты,
  которые видел ИИ); кнопка «пожаловаться на разбор» → панель оценки; дисклеймер на каждом отчёте.

## M5. Мультишкольность (ТЗ-4, «разные школы»)

- Переключатели на уровне расчётного профиля (см. M0) — все модули (карты, транзиты, разборы)
  уважают выбранный пресет.
- Вики-статьи о школах ([M7](#m7-база-знаний-тз-6)) связаны с пресетами: «посмотреть мою карту в этой традиции».
- Дорожная карта школ — [10-выводы, §4](../strategy/10-выводы-и-позиционирование.md).

## M6. Нумерология и минералы (ТЗ-5)

- **Калькуляторы** (все детерминированные, без ИИ в расчёте): Матрица судьбы (октаграмма,
  арканы, карта здоровья по чакрам, 4 предназначения, совместимость двух дат), психоматрица
  Пифагора, число жизненного пути/судьбы/души/имени (зафиксированная таблица кириллицы —
  документируется в вики), персональный год/месяц/день, совместимость по числам.
- **Зафиксированные конвенции** (публикуются в вики для прозрачности): метод редукции —
  сквозной с сохранением 11/22; таблица кириллицы — циклическая 1–9; правила для Ё/Й описаны.
- **Каталог камней** (M6b): карточка камня (свойства, соответствия знакам/планетам/декадам/
  арканам/чакрам, «кому подходит»), фильтры по знаку/цвету/назначению; кросс-ссылки из
  результатов расчётов («ваш аркан 3 — камни: …»). Витрина без своего e-commerce; партнёрские
  ссылки — этап 2.
- Монетизация: PDF-расшифровки матрицы/психоматрицы, детская матрица, совместимость.

## M7. База знаний (ТЗ-6)

- Разделы: планеты (10+узлы, Лилит, Хирон), знаки (12), дома (12), аспекты, стихии/кресты,
  школы и традиции, накшатры (27), арканы (22), лунные дни (30), камни, глоссарий.
- **Двойная роль**: SEO-статьи (перелинковка с калькуляторами и гороскопами) + **корпус RAG**
  для ИИ-помощника: у каждой статьи есть машиночитаемые «фрагменты интерпретаций»
  (планета×знак, планета×дом, аспект×пара — отдельные записи БД с эмбеддингами).
- Контент лицензионно чистый: собственные тексты (LLM-черновик + редактура) и синтез классики
  в общественном достоянии; запрет копирования защищённых корпусов (Лиз Грин, Хэнд, Сакоян).
- Редакционный процесс: черновик → ревью редактора → публикация; версии статей.

## M8. Коммьюнити (ТЗ-7)

- **Лента публикаций**: пост с прикреплённой картой (своей/анонимизированной) — формат
  «выложи карту — получи разбор» (главная живая UGC-механика астрофорумов рунета).
- **Сравнение карт с друзьями**: добавление в друзья, синастрия-виджет (viral loop Co-Star);
  приватность — карта видна только по явному разрешению.
- **Галереи**: подборки карт знаменитостей (встроенная база) с обсуждениями — UGC + SEO long-tail.
- Комментарии, реакции, репутация (уровни за качественные разборы), жалобы.
- **Модерация** (этический кодекс по образцу OPA/ISAR): запрещены предсказания смерти/диагнозов,
  «порча/привороты», медицинские и финансовые директивы; вероятностные формулировки;
  автомодерация LLM-классификатором + очередь ручной модерации; ПД-гигиена (данные рождения
  третьих лиц — только анонимизированно).

## M9. Монетизация и биллинг

- Подписка (месяц/год, триал 7 дней), разовые покупки PDF, внутренние «промокоды».
- Платёжные провайдеры: ЮKassa (первичный) / CloudPayments (резерв), СБП; чеки 54-ФЗ;
  вебхуки идемпотентны. Квиз-онбординг → персонализированный пейвол (паттерн Nebula).
- Email/push-цепочки: брошенный расчёт, окончание триала, сезонные кампании («ваш 2027 год»).

## M10. Маркетплейс консультаций (этап 2 — только архитектурный задел)

- Каталог верифицированных астрологов (профиль, школа, рейтинг, цены), чат-консультации
  пакетами (не поминутная тарификация «в тумане» — антитеза репутационным проблемам Astro7),
  эскроу-логика, честные возвраты.
- Юридическая схема: агентский договор, исполнители — самозанятые/ИП, автопроверка статуса
  НПД через API ФНС, чек от исполнителя клиенту (см. [05](../research/05-рынок-услуги-юридика.md), §4).
- В MVP: статусная модель заказов (M3) и роль `expert` в модели данных уже предусмотрены.

## Пользовательские пути (ключевые)

1. **SEO → подписка**: «натальная карта онлайн» → калькулятор → карта+краткий разбор →
   «сохранить и получить полный разбор» → регистрация → квиз → пейвол → подписка/PDF.
2. **Матрица → PDF**: «матрица судьбы рассчитать» → калькулятор → бесплатная схема+краткие
   значения → PDF 490 ₽ → апселл «совместимость», подписка.
3. **Гороскоп → ежедневная привычка**: «гороскоп телец завтра» → страница → «получать мой
   персональный прогноз» → регистрация → Telegram-бот/push → ежедневный возврат.
4. **Индивидуальный прогноз**: кабинет → мастер заказа («переезд в сентябре?») → оплата →
   ИИ-отчёт PDF → (этап 2: «обсудить с астрологом»).

## Границы MVP

**В MVP**: M0, M1 (включая восточный годовой гороскоп), M2, M3 (ИИ), M4 (западная школа +
кармический слой: узлы, средняя Лилит, Селена как анти-Лилит, ретропланеты), M6 (матрица,
психоматрица, числа, камни-справочник), M7 (ядро статей — минимум для RAG и SEO — обязательно
в MVP), M9 (подписка+PDF).
**Версия 1.x**: джйотиш (продуктовый модуль), ба-цзы, истинная Лилит, M8 коммьюнити, Telegram-бот.
**Версия 2**: M10 маркетплейс, хорар/электив, гамбургская/эллинистика, точки Глобы, партнёрская витрина камней.
