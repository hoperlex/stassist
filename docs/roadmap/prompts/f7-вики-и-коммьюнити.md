# Промт Ф7 — База знаний и коммьюнити

## Контекст
Фаза Ф7 портала «Зодиакум». Прочитай: `docs/architecture/20-архитектура-услуг.md` (M7, M8),
`docs/architecture/22-модель-данных.md` §2 (celebrities), §6–7 и §7б (notifications),
`docs/research/08-seo-и-коммьюнити.md`.
Примечание к границам MVP: вики-часть (M7) обязательна в MVP; коммьюнити-часть (M8) может
быть выполнена отдельной итерацией 1.x (см. 30-план-реализации).

## Задача
Вики как SEO-актив и источник RAG + социальный контур с безопасной модерацией.

## Требования — вики (M7)
1. Разделы и шаблоны статей: планеты (14: 10 + узлы, Лилит, Хирон), знаки (12), дома (12),
   аспекты (7 основных), школы (6 обзорных), лунные дни (30), арканы (22), глоссарий (50+).
   Статья = редакционный текст + машиночитаемые чанки (связь `interpretation_chunks.source_article_id`).
2. Производство: LLM-черновики по шаблонам → редакционная очередь (status) → публикация;
   версии статей; наполни стартовый набор: все знаки, все планеты, 12 домов, лунные дни, арканы.
   Программатический кластер «планета в знаке/доме» (матрица 23 §2, 240 URL): отдельные
   SSR-страницы `/planety/{planeta}-v-{znak}` и `/planety/{planeta}-v-{n}-dome`, генерируемые
   шаблоном из чанков Ф4, + хабы 10 планет с перелинковкой.
3. SSR `/wiki/{razdel}/{slug}` + хлебные крошки + перелинковка с калькуляторами («посчитать
   свою Венеру») и гороскопами; JSON-LD Article; sitemap-кластер.
4. Внутренний поиск (pg_trgm) по вики.

## Требования — коммьюнити (M8)
1. **Публикация карты**: пост kind='chart_review_request' — прикрепляется **анонимизированная
   копия** карты (позиции/дома/аспекты, БЕЗ даты/времени/места/имени — генерация копии на
   бэкенде, тест на отсутствие ПД); заголовок-вопрос; лента с фильтрами (новые/без ответа/
   популярные).
2. Комментарии (дерево 2 уровня), реакции, репутация (+баллы за отмеченный автором «полезный
   разбор», бейджи).
3. **Друзья и синастрия**: заявка → подтверждение → опция «поделиться картой» (явное согласие,
   отзываемое) → виджет синастрии пары в кабинете; без согласия карта друга недоступна.
4. **Галереи знаменитостей**: страницы `/karta/{celebrity}` из базы `celebrities` (наполни
   100 персон с публичными данными рождения и указанием источника/рейтинга достоверности) +
   обсуждение под каждой.
5. **Модерация**: премодерация первых 3 постов новичка; автоклассификатор (LLM) на запрещённое
   (предсказания смерти/болезней, «порча/привороты», медицинские/финансовые директивы,
   оскорбления) → очередь модератора (роль moderator, интерфейс: лента, жалобы, действия);
   правила сообщества — страница (этический кодекс по мотивам OPA/ISAR, на русском);
   `reports_ugc` — жалобы.
6. Уведомления: ответ на пост/комментарий, заявка в друзья (in-app + e-mail дайджест).

## Верификация
- Тест анонимизации: JSON опубликованной карты не содержит ПД-полей (автотест по схеме);
  попытка доступа к карте друга без согласия → 403.
- E2E: пост с картой → комментарий-разбор → отметка «полезно» → репутация выросла;
  жалоба → пост скрыт в очередь модератора → решение.
- Модерация: 20 провокационных постов (список в тесте) — ≥18 перехвачены автоклассификатором,
  остальные — вручную из очереди.
- Вики: 100% статей стартового набора опубликованы, чанки связаны; SSR + JSON-LD валидны;
  поиск находит «Венера в Скорпионе».
- Выборочная вычитка 10 вики-статей: фактические ошибки астрологической матчасти отсутствуют
  (сверка с 01-школы-астрологии.md и классическими соответствиями).
