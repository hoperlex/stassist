# Промт Ф6 — Нумерология: PDF-продукты и каталог камней

## Контекст
Фаза Ф6 портала «Stassist». Прочитай: `docs/architecture/20-архитектура-услуг.md` (M6),
`docs/architecture/22-модель-данных.md` §3 (orders, статусная модель) и §6 (stones),
`docs/research/06-нумерология-и-минералы.md`. Калькуляторы уже есть (Ф3), LLM-конвейер — (Ф4).
⚠ Юридически: название «Матрица судьбы» — заявленный товарный знак школы Ладини; до
коммерческого запуска продукта обязательна проверка в Роспатенте (чек-лист Ф8) — при
необходимости используй нейтральный нейминг продукта в UI.

## Задача
Превратить нумерологические калькуляторы в продаваемые продукты и построить каталог камней.

## Требования
1. **PDF «Матрица судьбы — полная расшифровка»** (главный разовый продукт ниши):
   октаграмма-SVG + все позиции (личные качества, линия отношений, денежный канал, карта
   здоровья по чакрам, 4 предназначения, возрастные периоды) — тексты собираются из чанков
   `arcanum×position` (детерминированная сборка) + LLM-связки между разделами (конвейер Ф4);
   25–35 страниц, оглавление, дисклеймер. Варианты: «детская матрица», «совместимость двух дат».
2. **PDF «Психоматрица»** и **«Нумерологический профиль»** (число пути + имя + циклы) —
   та же механика, 10–15 страниц.
3. **Заказы**: `orders` (kind='pdf_report') со статусной моделью из док. 22 §3; до Ф8 оплата —
   тестовый провайдер-заглушка за флагом (демо-режим), архитектура готова к ЮKassa.
4. **Каталог камней** (`stones`): наполни 60–80 карточек (worker-скрипт + LLM-черновики по
   шаблону из открытых источников с рерайтом, минералогические факты — по справочным данным;
   магические/лечебные свойства — с формулировками «по традиции считается» и дисклеймером);
   соответствия: знаки, планеты, декады, арканы, чакры. Страницы `/kamni/{kamen}`,
   `/kamni-po-znaku/{znak}`, фильтры (знак/цвет/назначение).
5. **Кросс-связки**: в результатах калькуляторов и PDF — блок «ваши камни» (по знаку/аркану/
   числу); на страницах камней — «проверьте свой аркан» (виджет). Без e-commerce; поле
   `affiliate_url` в схеме — задел этапа 2.
6. **Персональные циклы** (кабинет): виджет «ваш персональный год/месяц/день» с кратким
   текстом из чанков — ежедневный контент для возврата.

## Верификация
- Детерминизм: расчёты матрицы/психоматрицы для 10 контрольных дат совпадают с фикстурами Ф1.
- PDF: генерация ≤30 с, корректная кириллица, оглавление кликабельно, все разделы заполнены
  (нет «пустых арканов»), дисклеймер на месте; ручная вычитка 2 полных PDF.
- E2E: калькулятор матрицы → CTA → заказ (демо-оплата) → PDF в кабинете и по ссылке из письма.
- Каталог: SSR-страницы камней с JSON-LD BreadcrumbList; фильтры работают без JS-ошибок;
  выборочная проверка 10 карточек на фактические ляпы (минералогия) и наличие «по традиции».
- Кросс-блоки: аркан → камни → аркан замыкается корректными ссылками.
