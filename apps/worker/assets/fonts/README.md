# Шрифты PDF-отчётов

`DejaVuSans.ttf` / `DejaVuSans-Bold.ttf` — шрифт **DejaVu Sans** (проект DejaVu Fonts,
https://dejavu-fonts.github.io/), производный от Bitstream Vera Fonts с добавленным покрытием
кириллицы. Лицензия — свободная (Bitstream Vera License + общественное достояние дополнений
DejaVu), явно разрешает встраивание в документы и редистрибуцию без отдельного разрешения; полный
текст лицензии — на сайте проекта.

Используется `apps/worker/src/pdf/fonts.ts` для встраивания в генерируемые PDF (см.
docs/roadmap/prompts/f6-нумерология-и-камни.md, верификация «корректная кириллица») — базовые 14
PDF-шрифтов (Helvetica и т.п.), которые `pdfkit` использует по умолчанию, кириллицу не
поддерживают.
